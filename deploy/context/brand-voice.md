# 한국 상세페이지 카피 & 톤 가이드

## 카피 패턴

### 문제-해결 구조 (가장 효과적)
1. 공감 (이런 고민 있으셨나요?)
2. 문제 심화 (계속 방치하면...)
3. 해결 제시 (그래서 만들었습니다)
4. 증거 (98% 만족도, 시험성적서)
5. 행동 촉구 (지금 바로)

### 수치 강조 (신뢰도 확보)
- "131.60% 흡수율" (정확한 소수점)
- "10배 영양소 보존"
- "98% UV 차단"
- "연간 120만원 절감"
- "36년 역사"

### 넘버링 패턴 (56% 사용률)
- Point 01. / Point 02. / Point 03.
- STEP 1 / STEP 2 / STEP 3
- 01 특장점 / 02 특장점 / 03 특장점

### 인증/수상 강조
- 비건/무자극 인증 뱃지
- 시험성적서/특허 이미지
- KC, ISO, KSA 마크
- 수상 트로피 아이콘

## 톤별 스타일

### trust (신뢰)
- "검증된", "인증받은", "시험성적서 확인"
- 수치 데이터 + 인증 마크 강조
- 차분한 네이비/그린 톤
- 대상: 건강기기, 산업재, 전자기기

### premium (프리미엄)
- "특별한", "엄선된", "장인의 손길"
- 골드 액센트 + 크래프트 텍스처
- 영문 스크립트 폰트 + 한글 병기
- 대상: 전통주, 화장품, 인테리어

### friendly (친근)
- "쉽게", "간편하게", "이것만 알면"
- 밝은 톤, 부드러운 라운드 디자인
- 사용 장면 사진 (모델 활용)
- 대상: 식품, 생활용품

### trendy (트렌디)
- "MZ세대 필수", "요즘 핫한"
- 채팅 버블 리뷰, SNS 스타일 레이아웃
- 비비드 컬러 + 깔끔한 타이포
- 대상: 뷰티, 캐릭터 굿즈

## AI slop 회피

절대 사용 금지 표현:
- "혁신적인 솔루션"
- "완벽한 조화"
- "새로운 패러다임"
- "최적의 경험"
- "시너지를 극대화"
- "원스톱 솔루션"

대신 구체적으로:
- "혁신적인 솔루션" -> "먹기만 하면 되는 간편함"
- "완벽한 조화" -> "굴+미역의 감칠맛"
- "새로운 패러다임" -> "기존 제품 대비 10배 빠른 속도"
