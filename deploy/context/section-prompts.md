# 섹션별 Gemini 이미지 생성 프롬프트 가이드

Gemini로 각 섹션의 완성 이미지를 생성할 때 사용하는 레이아웃/스타일 가이드.
모든 이미지: 860px 폭, 한국어 텍스트 포함, 사실적 사진 스타일.

## 공통 스타일 앵커

- Canon 5D Mark IV / Sony A7R IV 품질
- 한국인 모델 (자연스러운 피부 텍스처)
- 프로 스튜디오 조명
- 참고: 아모레퍼시픽, 설화수 급 광고 퀄리티
- 절대 금지: 일러스트, 카툰, 벡터아트

## hooking / hero (800px 높이)

**레이아웃:** 대형 헤드라인 중앙, 긴박감 뱃지 우상단, CTA 버튼 하단 중앙
**배경:** Primary 컬러 -> 어두운 쉐이드 그라디언트
**무드:** 프리미엄, 신뢰감, 행동 유도
**텍스트:** 헤드라인(큰 볼드) + 서브헤드(작은 레귤러) + CTA 버튼
**카테고리 변형:**
- food: 따뜻한 톤, 식재료/접시 배경
- cosmetics: 깔끔한 화이트+골드, 모델/제품 클로즈업
- interior: 우드 텍스처, 영문 스크립트 헤드라인
- electronics: 다크 배경, 사용 장면 실사
- industrial: 강렬한 빨강/검정 그라디언트

## problem (600px 높이)

**레이아웃:** 섹션 인트로 상단, 3-4개 페인 포인트 카드 가로 배치, 감성 훅 하단
**배경:** 밝은 회색 (#F3F4F6) 또는 크림
**무드:** 공감, "이거 내 얘기다" 느낌
**텍스트:** 각 카드에 아이콘 + 짧은 문구
**디자인:** 부드러운 그림자, 라운드 카드, 아이콘은 라인 스타일

## solution (520px 높이)

**레이아웃:** 제품 사진 + 텍스트 좌우 분할 (6:4 비율)
**배경:** 밝은 배경 또는 부드러운 그라디언트
**무드:** 해결, 안도, 기대
**텍스트:** 큰 헤드라인 + 3-4줄 설명 + 핵심 수치 강조
**디자인:** 제품 사진에 부드러운 그림자, 깔끔한 레이아웃

## features (700px 높이)

**레이아웃:** 3열 아이콘+텍스트 그리드 (가장 흔함) 또는 Point 01/02/03 넘버링
**배경:** 밝은/어두운 교차 (짝수 섹션: 밝은, 홀수: 어두운)
**무드:** 정보 전달, 체계적
**텍스트:** 각 특징마다 아이콘 + 제목 + 1-2줄 설명
**디자인:** 그리드 간격 균일, 아이콘 크기 통일 (48-64px)

## detail (600px 높이)

**레이아웃:** 제품 클로즈업 + 상세 설명 텍스트
**배경:** 화이트 또는 크림
**무드:** 꼼꼼한, 전문적
**텍스트:** 상세 스펙이나 성분 설명
**디자인:** 확대 사진 + 콜아웃 라벨

## howto (600px 높이)

**레이아웃:** Step 1/2/3/4 넘버링 + 각 단계 사진
**배경:** 밝은 배경
**무드:** 쉬운, 친근한
**텍스트:** 각 스텝에 번호 + 간단 설명
**디자인:** 사진 그리드 (2x2 또는 1x4)

## social_proof (800px 높이)

**레이아웃:** 후기 카드 형태 또는 채팅 버블 (카카오톡 스타일)
**배경:** 밝은 톤
**무드:** 신뢰, 검증
**텍스트:** 인용구 + 사용자명 + 별점
**카테고리 변형:**
- cosmetics: 수치 데이터 (98% 만족도, 131% 흡수율)
- food: 실제 리뷰 텍스트 + 별점
- beauty_equipment: 채팅 버블 리뷰 (최신 트렌드)

## trust (500px 높이)

**레이아웃:** 인증마크/수상 뱃지 가로 나열
**배경:** 화이트 또는 밝은 회색
**무드:** 권위, 검증
**텍스트:** 인증명 + 날짜/기관
**디자인:** 뱃지/로고 일정 크기, 골드 액센트

## specs (500px 높이)

**레이아웃:** 좌정렬 표 형태 (50%) 또는 카드 내 중앙정렬
**배경:** 밝은 회색
**무드:** 정보, 실용적
**텍스트:** 항목명: 값 형태
**디자인:** 깔끔한 표, 얇은 구분선

## guarantee (500px 높이)

**레이아웃:** 중앙 아이콘 + 보증 텍스트
**배경:** 크림 또는 밝은 배경
**무드:** 안심, 신뢰
**텍스트:** 보증 기간/조건 + 교환/반품 안내

## event_banner (400px 높이)

**레이아웃:** 풀와이드 배너, 큰 텍스트 중앙
**배경:** 강렬한 컬러 (빨강, 노랑) 또는 그라디언트
**무드:** 긴박, 행동 유도
**텍스트:** 할인율/기간 + CTA

## cta (600px 높이)

**레이아웃:** 중앙 CTA 텍스트 + 큰 버튼
**배경:** Primary 컬러 또는 라이프스타일 사진
**무드:** 마무리, 행동 촉구
**텍스트:** "지금 구매하기" + 혜택 요약
**디자인:** 큰 라운드 버튼, 대비 강한 색상
